## Description
This package generates difference between two json/yaml files in 3 different formats in cli.

## Setup
```sh
$ git clone https://github.com/vyachka1001/php-project-lvl2.git

$ make install
```

## Run tests

```sh
$ make test
```

## Utility info

```sh
$ ./bin/gendiff -h
```
